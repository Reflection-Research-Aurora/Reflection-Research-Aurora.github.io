常见问题

要看课程等级。Level 1 面向零基础或仅有少量 Python 经验的学生，不要求提前学过编程。Level 2 需要具备编程基础，可来自 Level 1 或同等水平的自学。Level 3 需要通过评估，并要求较强的技术基础。如果还不确定孩子适合哪一级，可联系我们进行免费评估。

可以，但前提是学生已经具备扎实的 Python 基础，例如变量、循环、函数以及基础数据处理能力。我们会在评估中确认这一点。若基础不足却直接从Level 2开始，前几周会明显感觉吃力，最终的学习结果也会大打折扣。

所有课程均在线上进行。教学与项目运行在 Google Colab 或同类云端环境中，不需要本地安装。学生只需要浏览器和稳定的网络连接。这也意味着安大略省各地，甚至更远地区的学生都可以参加。

Level 1：每周约 3–4 小时（1.5 小时直播课 + 2 小时自学）。Level 2：每周约 6–7 小时（2 小时直播课 + 4–5 小时项目实践）。Level 3：每周约 8–10 小时（2 小时直播课 + 6–8 小时独立研究与写作）。这些都是比较真实的投入估计——若只是把它当作普通作业而不是认真投入，通常会在 Level 2 和 Level 3 很快落后。

会间接地帮助。我们的项目真正建立的是技术深度和学生对项目的真实掌控，这才是让申请文书有说服力的关键。完成 Level 2 或 Level 3 的学生会拥有可写、可讲的真实经历：处理过的数据集、调试过的模型、分析过的结果，以及从失败中迭代出来的经验。这正是大学录取评审最看重的内容。我们不代写申请文书，这违反学校要求，文书由学生自己完成。但通过我们的项目，学生会拥有更多有价值的内容可写。

不保证，我们也会在一开始就明确说明这一点。期刊投稿是努力争取的目标，而不是付费即可获得的结果。同行评审竞争激烈，结果取决于项目质量与范围。相比仓促投稿到可信度低的开放获取期刊，一个执行扎实、范围清晰、投稿至可信期刊的项目，对申请更有说服力。我们的目标包括影响因子在3-10的期刊，例如Bioinformatics，BMC Bioinformatics、PLOS ONE、Computers 以及 Frontiers 系列等以质量为录用标准的期刊。

Level 2的项目是领域内前沿机器学习项目。Level 3的项目则是“CS × 任意领域”。学生曾做过健康数据、环境科学、体育分析、社会系统等多个方向的项目。计算方法本身，例如数据处理、机器学习建模、研究写作，具有跨领域迁移性。我们鼓励学生选择与大学申请目标及个人真实兴趣相关的领域，因为这种连接会在申请文书和面试中自然体现出来。

Level 3 遵循明确的研究推进路径（16 周以上），有清晰的阶段里程碑，并以可信的外部成果为目标。Level X 则面向那些目标、时间安排或学术背景不适合标准结构的学生。例如申请时间特殊、跨多个方向同时规划，或同时需要科研指导与学术辅导的学生。如果你还不确定最适合哪个标准等级，Level X 也可以作为很好的切入点。

现实情况总会有变化。请尽早联系我们，我们会尽量协商出合理方案。对于 Level 3项目，订金因已预留导师时间而不予退还；剩余分期款项则会根据项目已推进的程度具体处理。
