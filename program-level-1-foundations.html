Level 1

Foundations

For students starting from zero or very light coding experience.

Class Schedule Ask About Fit

Overview

A guided starting point for future researchers

Foundations is designed for Grade 9-10 students with zero coding background or only a small amount of Python exposure. The goal is to remove the intimidation factor around programming and help students start thinking like problem-solvers.

Students learn by building, not by memorizing syntax in isolation. This includes both traditional coding practice and modern AI-assisted development, so students graduate with the habits of a working developer, not just a student who passed a quiz.

Schedule

A 10-week class structure

Weeks 1-3

Python basics, variables, control flow, and functions. Includes vibe coding (using AI coding assistants to build faster, while learning to read and validate what the AI produces).

Weeks 4-7

Data handling and visualization with Pandas and Seaborn. Introduction to machine learning concepts: how classification and regression models work, and hands-on practice with scikit-learn.

Weeks 8-10

A guided end-to-end mini-project: loading a real public health dataset, exploring it visually, training a baseline model, and explaining the result. Example: predicting cardiovascular disease risk from health indicators.

The program is paced as a weekly progression, combining instruction, guided practice, and project-based application.

Outcomes

What students build in Level 1

Students complete a guided end-to-end prediction project using a real public health dataset, their first experience applying code to a question that actually matters.

They leave with functional Python skills, familiarity with AI-assisted development workflows, and the ability to connect data, visualization, and model output into a story they can explain in plain language.

Foundations is the launchpad for students who plan to move into independent CS research, competition-style problem solving, or application-focused portfolio work, regardless of which domain they eventually pursue.
